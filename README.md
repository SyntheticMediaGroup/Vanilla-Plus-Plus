# Vanilla-Plus-Plus
A client focused on enhancing the vanilla experience without adding too much bloat.


# Installation
1. Install and login to Prism Launcher or MultiMC.
2. Click `Add Instance`, then `Import from ZIP`
3. Decide whether to use a standard or rolling release.
* Standard releases:
  * Will stay on the installed version permanently (e.g. it won't automatically update past 1.19.3).
* Rolling/Universal releases:
  * Will always update to the latest version. (1.19.2 would update to 1.19.3 automatically)
4. Copy and Paste one of the links into the text box:
  * [Standard release](https://github.com/SyntheticMediaGroup/Vanilla-Plus-Plus/releases/download/7.0.0/Vanilla-Plus-Plus_1.19.3-7.0.0.zip)
  * [Rolling/Universal release](https://github.com/SyntheticMediaGroup/Vanilla-Plus-Plus/releases/download/7.0.0/Vanilla-Plus-Plus_Universal.zip)
5. Select `Ok`
6. Probably rename the pack unless you're a monster.
7. Wait for the pack to be installed then launch the game
